Convert to HTML

Convert your project to static HTML and host anywhere.

  1. Settings

Settings

The following settings allow you to customise features of the convert, below is an example of the default you could use within your mixture settings file.

"convertHtml": {
  "extension": ".html",
  "path": null,
  "ignoreUrls": null,
  "relativePaths": false,
  "default": null
}
  • extension allows you to define the extension that will be output for the converted pages, default is ".html".
  • path Allows you to define an output path on disk, the default location is "converted-html" within the root of the project folder.
  • ignoreUrls accepts an array of URLs that should not be output.
  • relativePaths accepts either true or false, if true then all paths (links, styles, images, etc) within the pages including URLs will be converted to relative paths and extensions will be added to any internal links (extension based upon the extension set above). Setting to false will leave your project as is.
  • default if null then all templates will be directly converted, for example "my-page.liquid" will be converted to "my-page.html". If you set a default of "index" then "my-page.liquid" would become "my-page/index.html" (as long as that page did not already exist).
