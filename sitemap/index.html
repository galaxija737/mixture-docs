Autogenerated Sitemap

  1. Introduction

Introduction

When activated Mixture will automatically generate a sitemap based on the template directory for your project, view a screencast announcing this feature.

All changes will be immediatley updated in the sitemap and _sitemap.json will be created in the "model" directory.

When detected the sitemap data will be loaded and available from within the mixture model as mixture.sitemap. The sitemap will be in alphabetical order with the "Index" page at the top.

The following is an example of the data that will be output:

[
  {
    "key": "index",
    "name": "Index",
    "url": "/index",
    "directory": false
  },
  {
    "key": "about",
    "name": "About",
    "url": "/about",
    "directory": false,
    "children": [
      {
        "key": "jobs",
        "name": "Jobs",
        "url": "/about/jobs",
        "directory": false
      }
    ]
  }
}

We've created a special Liquid filter "navigation" which makes consuming this data a snap please read the template section for more information.

You can activate or disable sitemap generation via the mixture settings.
