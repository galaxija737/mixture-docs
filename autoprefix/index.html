Settings

Read more about autoprefixer.

Once activated all css files covered by "preprocessorLocations" will be processed and prefixed. Watch a demonstration.

Overview of Mixture settings:

  • active activates autoprefixer
  • sourceMap true or false activates sourcemaps
  • browsers the browsers you want to target in your project (can be null)

Example settings (which should be added to mixture.json in the root of the project):

"autoPrefixer": {
  "active": true,
  "sourceMap": false,
  "browsers": [
  	"last 1 version", 
  	"> 1%", 
  	"ie 8", 
  	"ie 7"
  ]
}
