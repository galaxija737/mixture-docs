Sass globbing, no setup required

Introduction

Sass globbing allows you to import many Sass or Scss files in a single import statement. Mixture has support out of the box, no setup required.

Sass

We've updated how this works in the latest version of Mixture, you need to activate globbing in the Mixture settings as per the following example: 

"sass": {
  "globbing": true
}

For a full list of the settings for Sass please see the preprocessor section.

Once activated you can start adding your import statements:

Import a folder of files:

@import "library/mixins/*";

If you have issues you can also try...

@import "library/mixins/**";

Import a tree of files:

@import "library/mixins/**/*";

Globbed files are sorted alphabetically before importing them and globs are always relative to the current file.

Compass

To use Globbing in Mixture with compass you just need to adjust the compass/config.rb in the root of your project and add the following line at the top of the file require 'sass-globbing'

Once thats done the same logic applies as detailed above in Sass.
