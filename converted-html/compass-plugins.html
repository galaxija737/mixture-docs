Compass Plugins
Use compass plugins with Mixture.

Introduction

The recommended way to use custom plugins with compass is to point Mixture at your own Ruby install. You can then install gems as required in the standard way.

Example: Installing Susy

The following example details how to install the Susy compass plugin.

  • If you are already knowledgable in Ruby and have it installed on your machine you can gem install these frameworks and then link them manually in the Mixture compass/config.rb in the root of your project.
  • If you don't have Ruby installed you'll need to do this first, please refer to the screencasts below.
  • Once the gems are installed you will need to find them on your system (see screencasts for examples of this) and grab the absolute path.
  • Once you have the path add it to the config.rb as a require, example: require 'Ruby200/lib/ruby/gems/2.0.0/gems/susy-1.0.7/lib/susy.rb'
  • Thats it, you can now call and use Susy from your Sass files.

You can also avoid using gems if you wish by downloading the Susy sass files to your project and simply importing them like any other sass file.

Please make sure you switch out the paths in the examples for your own.

Example: Installing ZURB Foundation

The following example details how to install the ZURB Foundation compass plugin.

  • If you are already knowledgable in Ruby and have it installed on your machine you can gem install these frameworks and then link them manually in the Mixture compass/config.rb in the root of your project.
  • If you don't have Ruby installed you'll need to do this first, please refer to the screencasts below.
  • Once the gems are installed you will need to find them on your system (see screencasts for examples of this) and grab the absolute path.
  • Once you have the path add it to the config.rb as a require, example: require 'Ruby200/lib/ruby/gems/2.0.0/gems/zurb-foundation-4.0.9/lib/zurb-foundation.rb'
  • Open up zurb-foundation.rb within it's gem folder and change the path to version.rb to an absolute path, example /Ruby200/lib/ruby/gems/2.0.0/gems/zurb-foundation-4.0.9/lib/foundation/version.rb
  • If you now cd into your project directory and then into the compass directory.
  • Please make sure the config.rb is setup correctly and then run compass install foundation, this should add all necessary files.
  • If you then open the project in Mixture and click view locally, and then update the url to /index.html you will see an example of styles and more.
  • You should now be up and running.

Please make sure you switch out the paths in the examples for your own.
