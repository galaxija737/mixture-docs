Tools & Services
Tools & 3rd party services to improve your life!

Introduction

The following details additional services built into Mixture. We are working on various integrations with services and tooling that we think will aid your development process and so this list will grow with future updates.

Convert a Mixture project to flat HTML

This feature is not available while in "simple" mode.

This will convert a Mixture project to straight HTML and drop it into the root of your project directory under a directory called "converted-html".

You can adjust the location of the "converted-html" directory within the Mixture settings.

BrowserStack, testing in the cloud!

Once you have added your BrowserStack API key into your online Mixture account you will be able to access one click tunneling via Mixture to your local project - we even live refresh any changes!

Here's some details to get you started...

1. To get your API key from BrowserStack, sign in to BrowserStack > Your account > Local testing and grab the "Local testing key". Add this to your Mixture account.

2. You can access your online Mixture account by using the avatar button in the application header chrome, we will be releasing web login for your account very soon. Once in your account, click on services to add / update your API key.

If you are running Mixture and your local server address is not localhost this can sometimes cause issues viewing on BrowserStack (it really depends on your system setup). If this happens update the project settings and force Mixture to run on localhost and try again. See more about Mixture settings.
