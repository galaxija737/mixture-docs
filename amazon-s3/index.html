Amazon S3

You can publish and host your converted (static) project on Amazon S3, Mixture makes it easy!

  1. Introduction
  2. Settings

Introduction

Mixture supports publishing to Amazon S3 for static Mixture projects.

To begin you simply need to create an Amazon AWS account (if you don't already have one) and then create a new bucket. It's best to name the bucket as the domain you wish to use (eg. mysite.com), make sure you give the correct access rights to the bucket and set it up in the preferences for static website hosting.

You can read more about the Amazon setup process from this great guid.

Settings

One you have setup your Amazon bucket, simply create a json file in the root of your project called .s3settings and add the following with your Amazon details:

  • accessKey your Amazon access key
  • secretKey your Amazon secret key
  • bucket your Amazon bucket

Example settings:

{
  "accessKey": "your-key",
  "secretKey": "your-secret-key",
  "bucket": "your-bucket",
  "region": "your-region"
}

Thats it, you're all set - simply convert to HTML within the Mixture tools and click the Amazon S3 button to publish!
